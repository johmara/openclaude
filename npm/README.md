# @johmara/openclaude

A full-featured terminal UI for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Quick start

```sh
npx @johmara/openclaude
```

## Prerequisites

The [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) must be installed and authenticated before running OpenClaude.

## Documentation

See the [main repository](https://github.com/johmara/openclaude) for full documentation, keybindings, themes, and configuration options.
